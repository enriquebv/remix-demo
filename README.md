# Marvel Span

## Features

- [] Username login (without password)
- [] Show list of heros from Marvel API: https://developer.marvel.com/
- [] Show detail of a hero from hero list
  - [] Mark hero as liked.
  - [] Set hero note (from 0 to 10)
  - [] Add a comment
- [] 3 Best heros showcase
  - 1 point per like
  - 2 points per comment
